Getting started

Basically, there are two ways to use JSkat:

  • To play offline against AI players
  • To play online, using JSkat as a GUI to the International Skat Server (ISS)

Playing offline

To play against two computer AIs, you have to create a skat table and start a skat series at that table. Either use the menu or the corresponding button to create a skat table. You can name your table, if you like. (This is only necessary if you want to create more than one table at a time.)
Once the table has been created, you will find a button at the center of the table to start the skat series. In the dialog you can set names for the AI players and yourself and you can choose the type of AI player you want to play against.

Algorithmic
This player has a set of rules defined, which decide what card to play in various situations. Unfortunately, this player is far from fully developed. So it might satisfy an occasional player, who just wants to play a short game. To a more experienced and demanding player, the algorithmic player will seem rather crude and stupid.
